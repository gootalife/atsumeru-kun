# discord.js TypeScript テンプレート

[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy)

Visual Studio Code を使用することを推奨します。

## スクリプト

### start

`npm run compile` を先に実行しないとエラーが出るよ！

```bash
npm start
```

### build

ビルドします。

```bash
npm run build
```

### lint

静的検証ツール(ESLint)を使って問題を調べる。

```bash
npm run lint
```

### lint:fix

問題を修正する。

```bash
npm run lint:fix
```
